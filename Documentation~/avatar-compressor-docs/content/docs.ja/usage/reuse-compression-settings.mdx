このチュートリアルでは、カスタムプリセットを作成し、複数のアバター間で圧縮設定を共有する方法を説明します。

## カスタムプリセットを使うタイミング

カスタムプリセットは以下の場合に便利です：

- 複数のアバターに同じ設定を適用したい
- プラットフォーム別の設定を作りたい（PC vs Quest）
- 他のクリエイターと設定を共有したい
- ビルトインプリセット以上の細かい調整をしたい

## カスタムプリセットを作成する

<Steps>
<Step>
### 設定を調整

1. アバターに **LAC Texture Compressor** を追加
2. **Custom** プリセットを選択
3. 満足のいく結果になるまで設定を調整

<Callout type="info">
保存前に **Preview Compression Results** で設定を確認しましょう。
</Callout>
</Step>

<Step>
### プリセットとして保存

1. プリセットフィールドの横にある **+** ボタンをクリック
2. 保存場所を選択（例：`Assets/Presets/`）
3. プリセットに名前を付ける（例：「MyPCSettings」）

現在の設定が再利用可能なアセットとして保存されます。

</Step>

<Step>
### Menu Path を設定

Project ウィンドウで新しいプリセットアセットを選択し、以下を設定：

| フィールド      | 例               | 説明                       |
| --------------- | ---------------- | -------------------------- |
| **Menu Path**   | `My Presets/PC`  | ドロップダウンでの表示場所 |
| **Menu Order**  | `100`            | 小さい値ほど先に表示       |
| **Description** | `PC用高品質設定` | プリセット選択時に表示     |

<Callout type="info">
Menu Path が設定されているプリセットのみがドロップダウンに表示されます。
</Callout>
</Step>

<Step>
### 他のアバターで使用

1. 別のアバターに **LAC Texture Compressor** を追加
2. **Custom** プリセットを選択
3. **Custom Preset** ボタンをクリックし、保存したプリセットを選択

設定が自動的に適用されます。

</Step>
</Steps>

## よくあるユースケース

<Tabs items={['PC と Quest', '複数アバター', '共有']}>
<Tab value="PC と Quest">
**プラットフォームごとに別のプリセットを作成。**

| プリセット名    | Menu Path         | 設定                             |
| --------------- | ----------------- | -------------------------------- |
| PC High Quality | `Platforms/PC`    | Quality プリセット + 高解像度    |
| Quest Optimized | `Platforms/Quest` | Aggressive プリセット + ASTC形式 |

ビルド対象プラットフォームに応じて切り替えます。

</Tab>

<Tab value="複数アバター">
**アバターコレクション全体で一貫した設定を共有。**

1. 好みの設定でプリセットを作成
2. すべてのアバターに適用
3. プリセットを一度更新すれば、すべてのアバターに反映

プリセットへの変更は、リンクされたすべてのアバターに自動的に適用されます。

</Tab>

<Tab value="共有">
**他のクリエイターにプリセットを配布。**

1. プリセットを作成・設定
2. **Lock** を有効にして誤った変更を防止
3. `.asset` ファイルを共有

受け取った人はインポートしてすぐに使用できます。

</Tab>
</Tabs>

## ビルトイン Plus プリセットを使う

Avatar Compressor には、より細かい制御が可能な中間プリセットが含まれています：

| プリセット      | 使用するタイミング                              |
| --------------- | ----------------------------------------------- |
| **Quality+**    | Quality が強すぎて、High Quality が軽すぎる場合 |
| **Balanced+**   | Balanced が強すぎて、Quality が軽すぎる場合     |
| **Aggressive+** | Aggressive が強すぎて、Balanced が軽すぎる場合  |

<Steps>
<Step>
### Plus プリセットにアクセス

1. コンポーネントで **Custom** プリセットを選択
2. **Custom Preset** ボタンをクリック
3. **Built-in** カテゴリ内にあります

</Step>
</Steps>

## Tips

- **重要なプリセットをロック**: 誤って変更されたくないプリセットは **Lock** フィールドを有効にしましょう。
- **Menu Path で整理**: `PC/Quality` や `Quest/Optimized` のようなフォルダ形式のパスでプリセットを整理できます。
- **説明を付ける**: 各プリセットの用途を覚えておくために **Description** を追加しましょう。

これらの設定の詳細は [Texture Compressor リファレンス](../../components/texture-compressor#custom-presets)を参照してください。

## トラブルシューティング

<Accordions>
<Accordion title="プリセットがドロップダウンに表示されない">
- プリセットアセットに **Menu Path** が設定されているか確認
- プリセットファイルが存在し、破損していないか確認
- Inspector を閉じて再度開いてみる
</Accordion>

<Accordion title="プリセットを編集できない">
  プリセットが制限されている可能性があります： - **ロック済み**:
  プリセットアセットの Lock フィールドを無効化 - **ビルトイン**:
  読み取り専用です。**Edit** → **Unlink** で設定をコピー - **外部パッケージ**:
  ビルトインと同様。Unlink してローカルで編集
</Accordion>

<Accordion title="変更が保存されない">
- **Save** をクリックしてリンクされたプリセットに変更を書き込む
- プリセットが制限されている場合、変更はローカルにのみ適用
- プリセットに影響を与えずにローカル変更を保持したい場合は **Unlink** を使用
</Accordion>
</Accordions>
